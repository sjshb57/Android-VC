## B (16)  
- **API 级别**: 36  
- **分布比例**: 7.4%  
- [官方文档](https://developer.android.com/about/versions/16)  
- **最后更新**: 2025年12月1日  

### 功能列表
- **用户体验与系统UI**  
  以进度为中心的通知；更丰富的触觉反馈API；预测性返回手势更新和默认迁移/选择退出要求（针对API 16+）；自动主题化应用图标；移除全屏模式选择退出（针对API 16+）；弃用干扰性无障碍播报；动态壁纸的内容处理  
- **安全与隐私**  
  改进对Intent重定向攻击的防护；Android Keystore的密钥共享API；Android隐私沙盒更新；锁定MediaStore版本（针对API 16+）；更安全的Intent解析机制（针对API 16+）；GPU系统调用过滤（针对API 16+）  
- **相机与媒体**  
  UltraHDR图像增强（HEIC）；精确的颜色温度和色调调整；混合自动曝光模式；动态照片捕获Intent动作；相机夜景模式场景检测（EXTENSION_NIGHT_MODE_INDICATOR）；照片选择器改进（可嵌入、云搜索）  
- **核心功能**  
  ART内部更改和性能更新；JobScheduler配额优化和新的作业被放弃原因（STOP_REASON_TIMEOUT_ABANDONED）；16 KB页面大小兼容模式；有序广播优先级范围限制在同一进程内；固定速率作业调度优化（针对API 16+）  
- **连接**  
  通用测距API（RangingManager）；具有增强安全性的测距（Wi-Fi 6 802.11az）；配套设备管理器的设备状态API；改进的蓝牙配对丢失处理以及针对API 16+的新Intent和API  
- **无障碍**  
  改进的无障碍 API；通过 LEA 助听器将手机用作语音通话的麦克风输入；LEA 助听器的环境音量控制
- **图形**
  使用 AGSL 创建自定义图形效果（RuntimeColorFilter， RuntimeXfermode）
- **国际化**
  垂直文本渲染支持；区域偏好设置中的测量系统自定义
- **设备形态**
  针对电视的标准化画面和音频质量框架（MediaQuality 包）；虚拟设备所有者覆盖；自适应布局（平台忽略针对 API 16+ 的屏幕方向和宽高比限制）
- **健康与健身**
  过渡到细粒度的健康和健身权限（针对 Health Connect 的 android.permissions.health）（针对 API 16+）
